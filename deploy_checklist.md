# Deployment Checklist

## Before Deployment
- [ ] Run all tests
- [ ] Check database migrations
- [ ] Update requirements.txt
- [ ] Backup database
- [ ] Set maintenance mode ON

## Deployment
- [ ] Git pull latest changes
- [ ] Install dependencies
- [ ] Run migrations
- [ ] Clear cache
- [ ] Restart services

## After Deployment
- [ ] Verify site loads
- [ ] Check error logs
- [ ] Test critical features
- [ ] Set maintenance mode OFF
- [ ] Monitor performance
